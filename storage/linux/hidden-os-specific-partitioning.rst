3. *If this is the first time you unlock the hidden volume, you may need to create a partition on the hidden volume. You will need to open a partition manager such as* `GParted <https://gparted.org/>`_ *and create one or more partitions manually. Make sure to create the partitions on the device that appeared when unlocking the hidden volume.*
