FAQ sul software

Q:

 Che tipo di chiave pubblica è prevista per l'accesso SSH?
   Ci si aspetta una chiave (pubblica) in stile openssh. A seconda del
   sistema operativo ci sono diversi modi per determinare la chiave
   pubblica. Uno potrebbe essere "ssh-add -L", un altro potrebbe
   essere "cat ~/.ssh/id_rsa.pub". Se state usando Putty, consultate
   la ` documentazione di Putty`_.

Q:

 Il sistema operativo può essere esteso o configurato manualmente?
   Sì, è possibile impostare l'accesso ssh attraverso l'applicazione
   NextBox Nextcloud. In seguito potrai accedere al tuo NextBox usando
   ssh e potrai fare con il sistema quello che vuoi. Ovviamente
   forniremo supporto solo per la configurazione stock di NextBox.

Q:

 Dove posso vedere un changelog per le versioni di NextBox?
   Attualmente solo qui:

   Launchpad NextBox

Q:

 Come posso aggiornare il sistema operativo?
   Non c'è bisogno di aggiornare il sistema operativo NextBox a mano.
   Questo viene fatto usando "unattended-upgrades" e sopra il

   nextbox-daemon

    controllerà gli aggiornamenti manualmente (come soluzione di
   ripiego ridondante) ad ogni avvio. Siete liberi di aggiornare il
   vostro sistema operativo a mano usando "apt", se sapete cosa state
   facendo.

Q:

 Il NextBox ha un sistema operativo a 32bit o 64bit?
   Il sistema operativo usato è un sistema a 64 bit. L'output "arch":
   *aarch64*.

Q:

 Quali utenti e password di default sono creati di default sul sistema
operativo di NextBox's?
   Non c'è una sola password di default impostata su nessun utente (di
   sistema). L'unico utente non di sistema creato si chiama
   "nextuser", senza una password predefinita. L'unico modo per
   accedere a NextBox è usare "ssh" con una chiave pubblica, che può
   essere impostata tramite la NextBox App.
