Aggiornamento del sistema

Procedura dopo un aggiornamento del sistema  Mismatch di hash di avvio
La voce di avvio è cambiata

NitroPad osserva le modifiche ai file di sistema importanti per
rilevare la manomissione da parte di terzi. Questo significa che a
volte sono necessari passi speciali quando si aggiorna il sistema.
Questo può essere spesso riconosciuto dal fatto che il sistema ti
chiede di riavviare.

Procedura dopo un aggiornamento del sistema

Mismatch di hash di avvio

Se riavviate NitroPad dopo un aggiornamento, la schermata di avvio
dovrebbe inizialmente apparire come al solito senza un messaggio di
errore.


Se poi premete Invio, otterrete un messaggio di errore che assomiglia
a questo o simile:


Confermate il messaggio con Invio per far verificare i file modificati
dall'aggiornamento del sistema. Dopodiché vi verrà chiesta di nuovo
una conferma.


Ora inserite la Nitrokey se non è già così e confermate con Enter. Poi
dovete inserire il PIN utente della Nitrokey (default: 123456). Non
vedrete sullo schermo i caratteri che avete digitato.  Heads si
aspetta un layout di tastiera inglese, in cui per esempio "y" e "z"
sono scambiati rispetto al layout della tastiera tedesca. Se sei
d'accordo con la selezione di default (che è scritta in maiuscolo)
puoi anche semplicemente confermare premendo Invio.

   Please confirm that your GPG card is inserted [Y/n]:

   Verifying presence of GPG card...

   XXXXXXX: XXXXXXXXXX
   Please unlock the card

   Number: XXXX XXXXXX
   Holder:
   Counter: X
   PIN:

Ora NitroPad segna i file cambiati dall'aggiornamento come verificati.

La voce di avvio è cambiata

Poi, o appare un altro messaggio di errore o si avvia il sistema
operativo. Questo dipende dall'aggiornamento del sistema. Se appare il
seguente messaggio di errore, procedete come descritto di seguito:


Prima confermate il messaggio di errore mostrato sopra. Poi vediamo
una lista di tutte le possibili voci della barca - scegliete la prima
premendo semplicemente Invio.


Per salvare questo valore come predefinito, selezionate l'opzione
"Make Ubuntu the default" per Ubuntu o "Make Qubes the default" per
Qubes OS nella prossima finestra.


Ora la Nitrokey deve essere collegata, se questo non è ancora il caso.

   Confermate la domanda "Salvare un default modificherà il disco" con Y.
   Confermate la domanda "Volete aggiungere una crittografia del disco al TPM" con N.
   Confermate la domanda "Confermate che la vostra scheda GPG è inserita" con Y.
   Inserire il PIN utente della Nitrokey quando richiesto (default: 123456).

   Avvertimento: Heads si aspetta un layout di tastiera inglese, in
     cui per esempio "y" e "z" sono scambiati rispetto al layout della
     tastiera tedesca. Se sei d'accordo con la selezione di default
     (che è scritta in maiuscolo) puoi anche semplicemente confermare
     premendo Invio.

   Saving a default will modify the disk. Proceed? [Y/n]:

   Do you wish to add a disk encryption to the TPM [y/N]:

   Please confirm that your GPG card is inserted [Y/n]:

   Verifying presence of GPG card...

   XXXXXXX: XXXXXXXXXX
   Please unlock the card

   Number: XXXX XXXXXX
   Holder:
   Counter: X
   PIN:

Il sistema si avvierà quindi.
