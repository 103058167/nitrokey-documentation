Nitrokey 3 con Windows

Autenticazione senza password

Autenticazione a due fattori (2FA)


Nitrokey 3 supporta l'autenticazione a due fattori (2FA) e
l'autenticazione senza password:

Con autenticazione senza password, l'inserimento di una password è
sostituito dal login con la Nitrokey 3 e un PIN.

Con autenticazione a due fattori (2FA), la Nitrokey 3 viene
controllata in aggiunta alla password.

Nitrokey 3 può essere utilizzato con qualsiasi browser attuale.

Importante: L'App Nitrokey non può essere utilizzata per la Nitrokey
  3.

Autenticazione senza password

Apri una pagina web che supporti FIDO2 (attualmente solo Microsoft).

Accedi al sito web e vai su "Set up security key" nelle impostazioni
di sicurezza del tuo account.

Ora devi impostare un PIN per la tua Nitrokey 3.

Tocca il pulsante della tua Nitrokey 3 quando ti viene richiesto.

Una volta che hai configurato con successo il dispositivo, dovrai
attivare la tua Nitrokey 3 in questo modo ogni volta che accedi, dopo
aver inserito il tuo PIN.

Autenticazione a due fattori (2FA)

Aprire uno dei ` siti web che supportano FIDO U2F
<https://www.dongleauth.com/>`_.

Accedi al sito web e abilita l'autenticazione a due fattori nelle
impostazioni del tuo account. (Nella maggior parte dei casi troverete
un link alla documentazione del servizio web supportato su
dongleauth.com)

Registra la tua Nitrokey 3 nelle impostazioni dell'account toccando il
pulsante per attivare la Nitrokey 3. Dopo aver configurato con
successo il dispositivo, devi attivare la Nitrokey 3 in questo modo
ogni volta che accedi.
